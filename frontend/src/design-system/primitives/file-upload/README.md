# File Upload Component

Drag & drop file upload with glassmorphism styling. Progressive enhancement - works without JavaScript.

## Quick Start

```html
<!-- Upload Zone -->
<div class="file-upload-zone">
  <input type="file" class="file-upload-zone__input" id="file" multiple />
  <label for="file" class="file-upload-zone__label">
    <div class="file-upload-zone__icon">
      <i class="fas fa-cloud-upload-alt"></i>
    </div>
    <div class="file-upload-zone__text">
      <p class="file-upload-zone__title">Drag & drop files here</p>
      <p class="file-upload-zone__subtitle">or click to browse</p>
    </div>
  </label>
</div>

<!-- File List (auto-generated by JS) -->
<div class="file-upload-list"></div>
```

## JavaScript (Optional)

```javascript
import { initFileUpload } from './file-upload.js';

const uploader = initFileUpload(zone, {
  maxSize: 5 * 1024 * 1024, // 5MB
  allowedTypes: ['image/jpeg', 'image/png', 'application/pdf'],
  multiple: true,
  onFileAdd: (file) => console.log('Added:', file.name),
});
```

## Variants

### Sizes

- `file-upload-zone--compact` - Small zone
- Default - Standard zone
- `file-upload-zone--large` - Large zone

### States (JS)

- `file-upload-zone--dragover` - Active drag
- `file-upload-zone--uploading` - Upload in progress
- `file-upload-zone--success` - Upload complete
- `file-upload-zone--error` - Upload failed
- `file-upload-zone--disabled` - Disabled

## File List

Files show automatically with:

- Image previews or file type icons
- Name, size, upload progress
- Remove button
- State indicators (pending, uploading, success, error)

### File States

- `file-upload-item--pending` - Waiting
- `file-upload-item--uploading` - Uploading (shows progress bar)
- `file-upload-item--success` - Complete
- `file-upload-item--error` - Failed (shows error message)

## Use Cases

- **Document Upload** - PDF, Word, Excel
- **Image Upload** - Photos with previews
- **Profile Picture** - Single image upload
- **Bulk Upload** - Multiple files at once
